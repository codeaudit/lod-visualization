DaCena

Project overview

DaCENA (Data Context for News Articles) is a web application that showcases a new approach to reading online news articles with the support of a data context built from interlinked facts available in the Web of Data. Given a source article, a set of facts that are estimated to be more interesting for the readers are extracted from the Web and presented using tailored information visualization methods and an interactive user interface. By looking at these facts, the reader access background factual knowledge with the advantage of being supported in the interpretation of the news content and being suggested connections to related topics to further explore.

Interlinked Facts in the Web of Data

The Web has transformed into a large scale repository of facts. Facts represent features of specific entities. For example the fact < Brack Obama birthdate 1961-08-04> in DBPedia - a data-based representation of Wikipedia content – describes the year when Barack Obama was born. Semantic associations are facts that represent relations existing between two entities. For example, FACTX-> represent that …. Several semantic associations that mention a common entity can be interlinked, forming a network of interlinked associations, or, interlinked facts. For example, if we also know that FACTY<-, we know that the entity subject_of_FACTX is connected to the entity subject of subject_of_FACTY because – natural language explanation of the association -.

Data Context for a News Articles as a Network of Interlinked Fact

DaCENA uses a novel interactive user interface to presents to the reader a set of interlinked facts that describe semantic associations between the main topic of an article, represented by an entity, and several other entities that are mentioned in the article. The interlinked associations represent context data that readers can navigate through with the goal to let them explore background knowledge relevant to the article by viewing at the network of relations that interconnect the entities in the article. DaCENA strive to show only those semantic associations that are evaluated to be more interesting for the readers, among a very large amount of existing ones. To this end DaCENA looks for serendipitous associations by combining two different criteria: relevance, which captures how related an association is to the content of the article, and unexpectedness, which captures how an association is likely to be unknown to the reader.
